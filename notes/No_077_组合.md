# LeetCode No.77 组合

## 题目链接: https://leetcode.cn/problems/combinations/description/
## 题目描述
给定两个整数 n 和 k，返回范围 [1, n] 中所有可能的 k 个数的组合。

你可以按 任何顺序 返回答案。


## 解题思路
- ** 方法 1 ** ：递归回溯
回溯三部曲：
1. 递归函数的返回值及参数
2. 回溯函数终止条件
3. 单层搜索的过程
想象成一个树形结构，树的宽度是for循环的宽度。深度是递归。
1. 递归函数的返回值及参数：传入的参数有n,k,path,result,startIndex, n和k是题目给的，path是记录访问过的路径。result是存放最后的结果，startIndex是从哪个点开始遍历。
2. 终止条件：如果path的路径的数量和k相等，那么就加入result中。然后返回
3. 单层的搜索逻辑：从startIndex开始，到n结束。然后把path加入进行，进行递归，然后进行回溯。
3 这里可以优化，如果n=4，k=3. 那么最开始的一层就最多能到2，因为2，3，4.但3就到不了了。所以是n-(k-len(path))+2.其中k-len(path)是还可以选几个数字。n-(k-len(path))可以选到的数字。然后+2是因为右闭。比如说第一层4-(3-0)+2 = 3也就是能取到2
时间复杂度`O(C(N,K))`  
空间复杂度：`O(K)`
