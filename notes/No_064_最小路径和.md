# LeetCode No.64 最小路径和

## 题目链接: https://leetcode.cn/problems/minimum-path-sum/description/

## 题目描述
给定一个包含非负整数的 m x n 网格 grid ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。
## 解题思路
- ** 方法 1 ** ：动态规划
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[j]的定义为：表示从(0,0)出发，到(i,j)的最小路径和。
2. 确定递推公式：
dp[j] 可以从dp[j] 和dp[j-1]两个方向到达，所以，我们只需要知道到dp[j]和dp[j-1]的路径就行。所以
dp[j] = min(dp[j], dp[j-1]) + grid[i][j]
3. dp数组如何初始化：
dp[0] = grid[0][0]
for (int j = 1; j < n; j++) dp[j] = dp[j - 1] + grid[0][j];
4. 确定遍历顺序：
从递推公式可以看出来，要根据从左到右，从上至下进行遍历。
5. 举例推导dp数组:
时间复杂度`O(M*N)` 
空间复杂度：`O(N)`
