# LeetCode No.718 最长重复子数组

## 题目链接: https://leetcode.cn/problems/maximum-length-of-repeated-subarray/description/

## 题目描述
给两个整数数组 nums1 和 nums2 ，返回 两个数组中 公共的 、长度最长的子数组的长度 。
## 解题思路
- ** 方法 1 ** ：动态规划
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i][j]的定义为：以i-1结尾的nums1，和以j-1结尾的nums2，的最长重复子数组。如果定义i，j结尾的话，那么下标0的情况不好定义。
2. 确定递推公式：
if nums1[i-1] == nums2[j-1]: dp[i][j] = dp[i-1][j-1] +1
因为定义是nums1 的下标i-1,nums2的下标j-1所以对比是否相同
3. dp数组如何初始化：
首先要初始化最上边的行，和最左边的列，也就是dp[i][0]和dp[0][j]均为0.因为i-1没有
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N^2)` 
空间复杂度：`O(N^2)`
