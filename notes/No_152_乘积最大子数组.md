# LeetCode No.152 乘积最大子数组

## 题目链接: https://leetcode.cn/problems/maximum-product-subarray/description/

## 题目描述
给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续 子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 32-位 整数。

请注意，一个只包含一个元素的数组的乘积是这个元素的值。


## 解题思路
- ** 方法 1 ** ：动态规划
这个题的核心思想是动态规划。我们需要维护两个变量，一个是当前子数组的最大乘积，另一个是当前子数组的最小乘积。因为负数可能会使得最小乘积变成最大乘积，所以我们需要同时维护这两个变量。
动态规划五部曲：
1. 定义状态：max_dp[i]表示以nums[i]结尾的子数组的最大乘积。min_dp[i]表示以nums[i]结尾的子数组的最小乘积。
2. 转移方程：
max_dp[i] = max(nums[i], max_dp[i-1] * nums[i], min_dp[i-1] * nums[i])，
min_dp[i] = min(nums[i], max_dp[i-1] * nums[i], min_dp[i-1] * nums[i])。
3. 初始化：max_dp[0] = nums[0]，min_dp[0] = nums[0]。
4. 遍历顺序：从前向后遍历。
