# LeetCode No.674 最长连续递增序列

## 题目链接: https://leetcode.cn/problems/longest-continuous-increasing-subsequence/

## 题目描述
给定一个未经排序的整数数组，找到最长且 连续递增的子序列，并返回该序列的长度。

连续递增的子序列 可以由两个下标 l 和 r（l < r）确定，如果对于每个 l <= i < r，都有 nums[i] < nums[i + 1] ，那么子序列 [nums[l], nums[l + 1], ..., nums[r - 1], nums[r]] 就是连续递增子序列。
## 解题思路
- ** 方法 1 ** ：动态规划
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i]的定义为：以i为截止的前面的所有子序列的最长递增子序列的长度为dp[i]
2. 确定递推公式：
dp[i] = dp[i-1]+1
因为是连续的，所以不需要对比每一个，只需要对比前一个即可。
3. dp数组如何初始化：
首先，整个数值应该初始化为1.因为单个的位置算1.
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N)` 
空间复杂度：`O(N)`
- ** 方法 2 ** ：滑动窗口
滑动窗口，right一直向右遍历，当不符合条件了，计算当前的长度。记住，在最后的时候也需要再次统计。因为有可能全部递增的情况，那么就导致无法统计。
时间复杂度`O(N)` 
空间复杂度：`O(1)`
