# LeetCode No.583 两个字符串的删除操作

## 题目链接: https://leetcode.cn/problems/delete-operation-for-two-strings/description/

## 题目描述
给定两个单词 word1 和 word2 ，返回使得 word1 和  word2 相同所需的最小步数。

每步 可以删除任意一个字符串中的一个字符。
## 解题思路
- ** 方法 1 ** ：动态规划 求出最长公共子序列，然后减去长度，求得需要进行的删除操作。
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i][j]的定义为：以i-1结尾的nums1，和以j-1结尾的nums2，的最长公共子序列。如果定义i，j结尾的话，那么下标0的情况不好定义。
2. 确定递推公式：
if nums1[i-1] == nums2[j-1]: dp[i][j] = dp[i-1][j-1] +1
因为定义是nums1 的下标i-1,nums2的下标j-1所以对比是否相同
如果不等于就取dp[i-1][j]和dp[i][j-1]取最大值，比如说ace和abc. ac和abc对比。ace和ab进行对比，取最大值。
3. dp数组如何初始化：
首先要初始化最上边的行，和最左边的列，也就是dp[i][0]和dp[0][j]均为0.因为i-1没有
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N^2)` 
空间复杂度：`O(N^2)`

- ** 方法 2 ** ：动态规划
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i][j]的定义为：以i-1结尾的nums1，和以j-1结尾的nums2，的最长公共子序列。如果定义i，j结尾的话，那么下标0的情况不好定义。
2. 确定递推公式：
if nums1[i-1] == nums2[j-1]: dp[i][j] = dp[i-1][j-1]不进行操作
因为定义是nums1 的下标i-1,nums2的下标j-1所以对比是否相同
else:
三种情况，nums1删除，nums2删除，nums1和nums2都删除，求最小值
min(dp[i-1][j]+1,dp[i][j-1]+1,dp[i-1][j-1]+2)
3. dp数组如何初始化：
初始化第一行，意味着，nums1是空的，如果要想要nums2匹配，就需要删除nums2所有，所以遍历第一行，初始值都是j
同理遍历第一列
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N^2)` 
空间复杂度：`O(N^2)`
