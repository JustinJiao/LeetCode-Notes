# LeetCode No.450 删除二叉搜索树中的节点

## 题目链接: https://leetcode.cn/problems/delete-node-in-a-bst/

## 题目描述
给定一个二叉搜索树的根节点 root 和一个值 key，删除二叉搜索树中的 key 对应的节点，并保证二叉搜索树的性质不变。返回二叉搜索树（有可能被更新）的根节点的引用。

一般来说，删除节点可分为两个步骤：

首先找到需要删除的节点；
如果找到了，删除它。
## 解题思路
- ** 方法 1 ** ：递归：
这道题的返回判断较为复杂。主要分为两种。第一种，没有找到key值。那么就直接返回NULL。第二种当前cur为key值，执行操作。分为四种情况：
1. 叶子节点，左右均为空，那么删除完什么都没有，直接返回None
2. 左子树不为空，右子树为空。那么就返回左子树。
3. 左子树为空，右子树不为空。返回右子树
4. 两个子树都不为空：先取cur的右子树为template。然后找到右子树的左节点，存放cur的左子树到右子树到左左叶子。然后返回cur的右子树
然后进入单层逻辑。如果key大于当前值，就找右子树。如果小于当前值就找左子树。最后return root
时间复杂度`O(H)` 
空间复杂度：`O(H)`
