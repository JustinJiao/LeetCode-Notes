# LeetCode No.114 二叉树展开为链表

## 题目链接: https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/description/

## 题目描述
给你二叉树的根结点 root ，请你将它展开为一个单链表：

展开后的单链表应该同样使用 TreeNode ，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null 。
展开后的单链表应该与二叉树 先序遍历 顺序相同。

## 解题思路
- ** 方法 1 ** ：迭代三部操作
1. 看看当前节点是否有左子树，如果没有，直接向右移动到下一个节点
2. 如果有左子树，那么找到左子树的最右边的节点
3. 将当前节点的右子树连接到左子树的最右边节点
4. 将当前节点的左子树移动到右子树的位置
5. 将当前节点的左子树置空
6. 向右移动到下一个节点
