# LeetCode No.155 最小栈

## 题目链接: https://leetcode.cn/problems/min-stack/description/

## 题目描述
设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

## 解题思路
- ** 方法 1 ** ：辅助栈
使用一个辅助栈来存储当前最小的元素。当我们将一个元素压入主栈时，如果这个元素小于或等于当前最小元素，我们也将它压入辅助栈。当我们弹出主栈的元素时，如果这个元素等于辅助栈的顶部元素，我们也弹出辅助栈的顶部元素。这样，辅助栈的顶部始终是当前主栈中的最小元素。
时间复杂度：`O(1)`，每个操作（push、pop、top、getMin）都在常数时间内完成。
空间复杂度：`O(n)`，最坏情况下，所有元素都被压入辅助栈。
