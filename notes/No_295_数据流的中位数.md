# LeetCode No.295 数据流的中位数

## 题目链接: https://leetcode.cn/problems/find-median-from-data-stream/

## 题目描述
中位数是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均值。

例如 arr = [2,3,4] 的中位数是 3 。
例如 arr = [2,3] 的中位数是 (2 + 3) / 2 = 2.5 。
实现 MedianFinder 类:

MedianFinder() 初始化 MedianFinder 对象。

void addNum(int num) 将数据流中的整数 num 添加到数据结构中。

double findMedian() 返回到目前为止所有元素的中位数。与实际答案相差 10-5 以内的答案将被接受。


## 解题思路
- ** 方法 1 ** ：大顶堆和小顶堆
使用两个堆来维护数据流中的元素，一个大顶堆用来存储较小的一半元素，一个小顶堆用来存储较大的一半元素。
当添加一个新的元素时，首先将其添加到大顶堆中，然后将大顶堆的堆顶元素移动到小顶堆中。接着，如果小顶堆的元素个数超过大顶堆的元素个数，则将小顶堆的堆顶元素移动到大顶堆中。这样可以保证大顶堆和小顶堆的元素个数之差不超过 1。
当需要找到中位数时，如果大顶堆和小顶堆的元素个数相等，则中位数为两个堆顶元素的平均值；如果大顶堆的元素个数多于小顶堆，则中位数为大顶堆的堆顶元素；如果小顶堆的元素个数多于大顶堆，则中位数为小顶堆的堆顶元素。
时间复杂度`O(log N)`  
空间复杂度：`O(N)`
