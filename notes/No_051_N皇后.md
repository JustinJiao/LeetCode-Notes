# LeetCode No.51 N皇后

## 题目链接: https://leetcode.cn/problems/n-queens/description/
## 题目描述
按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。


## 解题思路
- ** 方法 1 ** ：递归回溯
回溯三部曲：
1. 递归函数的返回值及参数
2. 回溯函数终止条件
3. 单层搜索的过程
想象成一个树形结构，树的宽度是for循环的宽度。深度是递归。
1. 递归函数的返回值及参数：row 代表是第几行 ，chessboard是棋盘状态。result是存放最后的结果。
2. 终止条件：如果到达叶子节点，也就是row到达了最后，那么就要记录这个棋盘的状态了。但这里需要注意加入的时候直接加入的一个string，避免浅拷贝的时候把里面的二维数组拷贝没了。
3. 单层的搜索逻辑：遍历每一列。也就是在当前行，遍历每一个位置，查看是否可以。如果可以就插入Q如果不行就继续找下一个位置。
isvalid需要判断三个，首先是这一列是否有Q。其次是看左上角是否有Q再看右上角是否有Q.

时间复杂度`O(N!)`  
空间复杂度：`O(N)`
