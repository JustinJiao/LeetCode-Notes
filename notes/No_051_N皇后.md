# LeetCode No.51 N皇后

## 题目链接: https://leetcode.cn/problems/n-queens/description/
## 题目描述
按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。


## 解题思路
- ** 方法 1 ** ：递归回溯
使用递归回溯的方法，逐行放置皇后。在放置每个皇后时，检查当前列、主对角线和副对角线是否已经有皇后存在。如果没有冲突，则继续放置下一个皇后；如果有冲突，则回退并尝试其他位置。最终，当所有皇后都成功放置时，将当前的棋盘状态加入结果列表中。
左上是row-col，右上是row+col，左下是row-col+n-1，右下是row+col
时间复杂度`O(N!)` 因为每一行都要放置一个皇后，第一行有n个选择，第二行有n-1个选择，第三行有n-2个选择，以此类推，所以总共有n*(n-1)*(n-2)*...*1 = n!种组合。
空间复杂度：`O(N)` 因为递归调用的深度最大为N，
空间复杂度: `O(N^2)` 因为需要一个N*N的棋盘来存储皇后的位置，所以空间复杂度是O(N^2)。
