# LeetCode  No.75 颜色分类

## 题目链接: # LeetCode  No.76 最小覆盖子串

## 题目链接: https://leetcode.cn/problems/minimum-window-substring/description/

## 题目描述
给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums ，原地 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

## 解题思路
- ** 方法 1 ** ：双指针（荷兰国旗问题）
这道题是经典的荷兰国旗问题。我们可以使用三个指针，low, mid, high。low指向0的下一个位置，mid用于遍历数组，high指向2的前一个位置。具体步骤如下：
1. 初始化low = 0, mid = 0, high = len(nums) - 1。
2. 当 mid <= high时，进行以下操作：
   - 如果nums[mid] == 0，交换nums[low]和nums[mid]，然后low和mid都加1。
   - 如果nums[mid] == 1，mid加1。
   - 如果nums[mid] == 2，交换nums[mid]和nums[high]，然后high减1。
3. 重复步骤2直到mid > high。
时间复杂度`O(N)`  
空间复杂度：`O(1)`
