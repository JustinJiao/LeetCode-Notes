# LeetCode No.093 复原IP地址

## 题目链接: https://leetcode.cn/problems/restore-ip-addresses/description/
## 题目描述
有效 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 '.' 分隔。

例如："0.1.2.201" 和 "192.168.1.1" 是 有效 IP 地址，但是 "0.011.255.245"、"192.168.1.312" 和 "192.168@1.1" 是 无效 IP 地址。
给定一个只包含数字的字符串 s ，用以表示一个 IP 地址，返回所有可能的有效 IP 地址，这些地址可以通过在 s 中插入 '.' 来形成。你 不能 重新排序或删除 s 中的任何数字。你可以按 任何 顺序返回答案。
## 解题思路
用startIndex来进行分割。startIndex = 0意味着在位置0后进行分割。当分割到最后一个的时候说明分割完毕啦。要分割完毕并且长度为4. 循环从startIndex开始循环到字符串结束。i是分割后的位置，startIndex是子串的开头，i是逐个分割。然后判断是否是回文子串。如果是才加入到path中。然后循环

时间复杂度`O(3^4）` IP地址最多包含4个数字，每个数字最多有3种可能的分割方式，则搜索树的最大深度为4，每个节点最多有3个子节点。
空间复杂度：`O(N)`
