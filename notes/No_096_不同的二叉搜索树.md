# LeetCode No.096 不同的二叉搜索树

## 题目链接: https://leetcode.cn/problems/unique-binary-search-trees/
## 题目描述
给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。
## 解题思路
首先了解到，二叉搜索树的性质。左子树都小于当前的根节点。右子树都大于当前节点。
对于二叉搜索树的种类是左子树的节点数量*右子树的节点数量。
所以对于i。可以让j从1到i分别为根节点。然后将他们的数量全部相加。比如说当j为2的话，i是3.那么j的左边就有节点1.也就是j-1个节点。右边是i-j个节点。
时间复杂度`O(N^2）` 
空间复杂度：`O(N)`
