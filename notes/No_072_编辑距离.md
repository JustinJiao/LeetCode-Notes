# LeetCode No.72 编辑距离

## 题目链接: https://leetcode.cn/problems/edit-distance/description/

## 题目描述
给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数  。

你可以对一个单词进行如下三种操作：

插入一个字符
删除一个字符
替换一个字符

## 解题思路

- ** 方法 1 ** ：动态规划
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i][j]的定义为：以i-1结尾的nums1，和以j-1结尾的nums2，的出现的个数。如果定义i，j结尾的话，那么下标0的情况不好定义。
2. 确定递推公式：
if nums1[i-1] == nums2[j-1]: dp[i][j] = dp[i-1][j-1]也就是什么都不操作
因为定义是nums1 的下标i-1,nums2的下标j-1所以对比是否相同。然后加上dp[i-1][j]也就是s可以回退一个，就是即使这个相同，我也不取他，看看前面有几个。
else: dp[i][j] = min(dp[i-1][j]+1,dp[i][j-1]+1,dp[i-1][j-1]+1)
这里分为三种情况：
删除：dp[i-1][j]+1 word1不匹配，那么就看删除这个字符，然后+1
增加：dp[i][j-1]+1 这个是不匹配，可以理解为增加了word2一个字符，所以就看word2这个字符前面是怎么样的，然后+1
替换：dp[i-1][j-1] +1 这个是替换，也就是两边都不要这个了替换成一个新的，所以+1

3. dp数组如何初始化：
初始化的话第一行为0就说吗是第一行没有，所以操作就是删除几个j，也就是j
第一列同理
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N^2)` 
空间复杂度：`O(N^2)`
