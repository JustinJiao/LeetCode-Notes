# LeetCode No.1035 不相交的线

## 题目链接: https://leetcode.cn/problems/uncrossed-lines/description/
## 题目描述
在两条独立的水平线上按给定的顺序写下 nums1 和 nums2 中的整数。

现在，可以绘制一些连接两个数字 nums1[i] 和 nums2[j] 的直线，这些直线需要同时满足：

 nums1[i] == nums2[j]
且绘制的直线不与任何其他连线（非水平线）相交。
请注意，连线即使在端点也不能相交：每个数字只能属于一条连线。

以这种方法绘制线条，并返回可以绘制的最大连线数。
## 解题思路
- ** 方法 1 ** ：动态规划 同1143一样。求最长公共子序列
动态规划五部曲：
1. 确定dp数组以及下标的含义：
dp[i][j]的定义为：以i-1结尾的nums1，和以j-1结尾的nums2，的最长公共子序列。如果定义i，j结尾的话，那么下标0的情况不好定义。
2. 确定递推公式：
if nums1[i-1] == nums2[j-1]: dp[i][j] = dp[i-1][j-1] +1
因为定义是nums1 的下标i-1,nums2的下标j-1所以对比是否相同
如果不等于就取dp[i-1][j]和dp[i][j-1]取最大值，比如说ace和abc. ac和abc对比。ace和ab进行对比，取最大值。
3. dp数组如何初始化：
首先要初始化最上边的行，和最左边的列，也就是dp[i][0]和dp[0][j]均为0.因为i-1没有
4. 确定遍历顺序：
从递推公式可以看出来,是从前向后遍历。
5. 举例推导dp数组:
时间复杂度`O(N^2)` 
空间复杂度：`O(N^2)`
